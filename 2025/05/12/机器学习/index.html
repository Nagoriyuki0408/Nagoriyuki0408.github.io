切换默认下载位置

1. 使用国内镜像源（推荐）

国内的 PyPI 镜像通常比默认的 PyPI 服务器快很多，尤其是像 TensorFlow 这样的大文件。你可以临时或永久使用国内源下载：

临时使用（仅本次安装生效）：

bash

复制编辑

pip install tensorflow -i https://pypi.tuna.tsinghua.edu.cn/simple

永久修改 pip 源（推荐）：

修改 pip 的默认下载源，使所有 pip install 都加速：

bash

复制编辑

pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

或者手动修改 pip 配置文件：

  • Windows：编辑 C:\Users\你的用户名\pip\pip.ini（如果不存在，就新建）：

    ini

    复制编辑

    [global] index-url = https://pypi.tuna.tsinghua.edu.cn/simple

  • Linux/macOS：修改 ~/.pip/pip.conf：

    ini

    复制编辑

    [global] index-url = https://pypi.tuna.tsinghua.edu.cn/simple

常见的国内镜像源：

镜像源 地址
清华 https://pypi.tuna.tsinghua.edu.cn/simple
阿里 https://mirrors.aliyun.com/pypi/simple
豆瓣 https://pypi.doubanio.com/simple
腾讯 https://mirrors.cloud.tencent.com/pypi/simple

2. 手动下载 .whl 文件

如果切换源仍然很慢，你可以手动下载 TensorFlow 的 .whl 文件，然后本地安装：

  1. 访问 https://pypi.tuna.tsinghua.edu.cn/simple/tensorflow/ 查找适合你的 Python 版本的 .whl 文件。

  2. 下载 .whl 文件到本地。

  3. 使用 pip install 安装：

    bash

    复制编辑

    pip install 路径/文件名.whl

试试看能否加速下载？如果有问题，可以贴出你的 Python 版本和操作系统，我帮你匹配正确的 TensorFlow 版本。

如果你想让 pip 在安装时下载到你电脑的指定目录，可以使用 --download（较老版本）或 --cache-dir 选项。

方法 1：指定 --cache-dir 存储下载文件

你可以使用 --cache-dir 选项让 pip 下载的文件存储到特定目录：

bash

复制编辑

pip install tensorflow --cache-dir C:\my_downloads

这样 pip 在安装 TensorFlow 时，会先将文件下载到 C:\my_downloads 目录，再进行安装。

方法 2：手动下载 .whl 文件

如果你想完全控制下载目录，可以手动下载 .whl 文件，然后本地安装：

  1. 访问 https://pypi.tuna.tsinghua.edu.cn/simple/tensorflow/

  2. 选择适合你的 Python 版本的 .whl 文件，手动下载到你的电脑某个目录，比如 D:\tensorflow\

  3. 运行：

    bash

    复制编辑

    pip install D:\tensorflow\tensorflow-2.x.x-cp39-cp39-win_amd64.whl

    （请替换 .whl 文件名为你实际下载的文件名）

方法 3：使用 pip download 仅下载不安装

如果你想 先下载再安装，可以用 pip download：

bash

复制编辑

pip download tensorflow -d C:\my_downloads

这样 TensorFlow 会下载到 C:\my_downloads 目录，但不会自动安装。然后你可以手动安装：

bash

复制编辑

pip install --no-index --find-links C:\my_downloads tensorflow

Jupyterbook

1. 安装 Jupyter Notebook

如果你尚未安装 Jupyter Notebook，可以通过以下方式进行安装：

  • 使用 Anaconda 安装：Anaconda 是一个集成了多种数据科学工具的发行版，其中包含了 Jupyter Notebook。

    • 下载 Anaconda：前往 Anaconda 官方网站 下载适用于你操作系统的安装包。

    • 安装 Anaconda：运行下载的安装包，按照提示完成安装。

  • 使用 pip 安装：如果你已经安装了 Python 环境，可以使用 pip 来安装 Jupyter Notebook。

    bash

    复制编辑

    pip install jupyter

2. 启动 Jupyter Notebook

安装完成后，你可以通过以下方式启动 Jupyter Notebook：

  • 使用 Anaconda Navigator：打开 Anaconda Navigator，点击 “Launch” 启动 Jupyter Notebook。

  • 使用命令行：打开命令提示符（Windows）或终端（macOS/Linux），输入以下命令启动：

    bash

    复制编辑

    jupyter notebook

启动后，默认浏览器会自动打开 Jupyter Notebook 的主界面。

3. 创建并运行 Python 代码

在 Jupyter Notebook 中，你可以通过以下步骤创建并运行 Python 代码：

  • 创建新笔记本：在 Jupyter Notebook 主界面，点击右上角的 “New” 按钮，然后选择 “Python 3” 创建一个新的笔记本。

  • 编写代码：在新建的笔记本中，你会看到一个单元格（cell）。在其中输入你的 Python 代码。例如：

    python

    复制编辑

    print("Hello, Jupyter!")

  • 运行代码：按下键盘上的 Shift + Enter，或者点击工具栏中的 “Run” 按钮，来执行单元格中的代码。输出结果会显示在单元格下方。

4. 加载并运行外部 Python 文件

如果你有现成的 .py 文件，想在 Jupyter Notebook 中运行，可以使用以下方法：CSDN

  • 使用 %run 魔法命令：在一个单元格中输入 %run your_script.py，然后运行该单元格即可执行 your_script.py 中的代码。

    python

    复制编辑

    %run your_script.py

请确保 your_script.py 文件位于当前 Jupyter Notebook 工作目录下，或者提供正确的文件路径。

  • 使用 %load 魔法命令：在一个单元格中输入 %load your_script.py，然后运行该单元格。这会将 your_script.py 中的代码加载到单元格中，你可以在编辑后再运行。

    python

    复制编辑

    %load your_script.py

加载后，单元格中会显示 your_script.py 的内容。你可以根据需要修改代码，然后再运行该单元格。
