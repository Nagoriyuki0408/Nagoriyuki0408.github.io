1：调用函数之前需要事前声明

函数之间互相调用。
3：递归调用：
函数自己调用自己。

▪ 函数的参数传递
形式参数是不占内存的，当调用函数的时候，才会给形参分配内存空间。
实参可以是常量，变量和表达式。
转化（隐含）：将实参转化为形参。
单向传值的作用，被调用函数只会处理形参，形参得到形参的值的时候，就切割了。
return语句只能返回一个值。
现在需要一个双向传值的事物，方法是传引用&。传引用作参数可以保障实参数据的安全。
▪ 引用类型
值传递：单向传递。
引用传递：双向传递。
引用类型、
引用（&）是标识符的别名
定义一个引用的时候，必须同时对它进行初始化，使它指向一个已存在的对象。
例如：
int i，j;
int &ri=i;//定义int引用ri，并初始化为变量i的引用。
j=10;
ri=j;//相当于把j赋值给i
一旦一个引用被初始化后，就不能改为指向其他对象
引用可以作为形参

▪ 内联函数
简单地说，就是在调用简单函数提高效率的工具。
减少调用函数的麻烦。
内联函数声明时的关键字 inline
编译时在调用处用函数体进行替换
节省了参数传递
控制转移等开销
注意：
内联函数中不能有循环语句和switch语句
内联函数的定义必须出现在内联函数在第一次被调用之前
例子：

return PIradiusradius;

return 0;

▪ 函数重载
函数多态性的体现，C++中允许功能相近的函数在相同的作用域内以相同函数名声明，从而形成重载。方便使用，便于记忆。
int add(int x,int y);
float add(float x,float y);
int add(int x,int y,int z);
注意事项，重载函数的形参必须不同：个数不同或者类型不同。
编译程序将根据实参和形参的类型及个数的最佳匹配来选择调用哪一个函数。
int add(int x,int y);
int add(int x,int y);
编译器不以形参名来区分
int add(int x,int y);
void add(int x,int y);
编译器不以返回值来区分
不要将不同功能的函数声明为重载函数，以免出现调用结果的误解，混淆，这样不好。
int add(int x,int y)
{return x+y;}
float add(float x,float y)
{return x-y;}
int sumOfSquare(int a,int b){
return aa+bb;
}
Double sumOfSquare(double a,double b){
return aa+bb;
}
